# AutoReconnect
A Vencord plugin to automatically reconnect to voice channels after a random delay when someone disconnects you.

> [!NOTE]
> While you're more than welcome to manually change the reconnect delay, I've added sliders for the min and max delay to be between 0.5s and 5s as anything below 0.5s might be too quick and flag your account and anything above 5s seemed too long.
